# Project Scope Statement: Personal Introduction Webpage

## Project Purpose
The purpose of this project is to create a professional personal landing page that introduces my background, academic goals, and interests, while practicing semantic HTML structure.

## Description
This project involves drafting a formal project scope statement and developing a single-page HTML document (`index.html`). The webpage will serve as a digital introduction and will be hosted via GitHub Pages, demonstrating foundational web development skills.

## Desired Results
- A clean, valid HTML5 document that renders correctly in modern browsers.
- Clear communication of my professional interests and academic goals.
- Successful application of project management "scoping" to ensure the project remains focused and is completed on time.

## Exclusions
- This project will not include external CSS frameworks (e.g., Bootstrap, Tailwind).
- This project will not include complex interactive JavaScript features.
- This project is limited to a single page; multi-page navigation is outside the scope of this assignment.
